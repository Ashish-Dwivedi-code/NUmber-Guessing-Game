# Number Guessing Game

A fun and interactive Number Guessing Game built using **HTML**, **CSS**, and **JavaScript**.

## About the Project

In this game, the computer randomly generates a number between **1 and 100**. The player has **10 attempts** to guess the correct number. After each guess, the game provides feedback indicating whether the guess is too high or too low. The game ends when the player guesses the correct number or runs out of attempts.

## Features

* Random number generation between 1 and 100
* 10 attempts to guess the number
* "Too High" and "Too Low" hints
* Win and Game Over messages
* Restart Game functionality
* Input validation
* Clean and responsive user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

##  How to Play

1. Enter a number between 1 and 100.
2. Click the **Guess** button.
3. Read the hint provided by the game.
4. Continue guessing until you find the correct number.
5. Use the **Restart Game** button to play again.
